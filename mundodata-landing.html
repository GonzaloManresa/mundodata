OpenAI lanza nueva API para análisis de datasets estructurados directamente desde Python

Esta semana fue movida en el mundo del data. OpenAI sorprendió con una integración directa con Pandas, Google anunció cambios en BigQuery y hay una nueva herramienta open-source de visualización que vale la pena conocer.

01

OpenAI + Pandas: análisis de datos con lenguaje natural

La nueva API permite hacer preguntas en español sobre un DataFrame y obtener resultados automáticamente. Esto cambia bastante el flujo de trabajo para analistas junior.

02

Google BigQuery reduce precios un 30% para queries bajo 1TB

Buena noticia para equipos chicos y freelancers. El cambio entra en vigor el 1 de mayo y aplica a todas las regiones.

03

Evidence.dev: reportes de datos con Markdown y SQL

Una herramienta open-source para crear dashboards estáticos directamente desde archivos .sql. Sin servidores, sin configuración.
