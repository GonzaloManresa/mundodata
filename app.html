OpenAI lanza nueva API para análisis de datasets estructurados directamente desde Python

Esta semana fue movida en el mundo del data. OpenAI sorprendió con una integración directa con Pandas, Google anunció cambios en BigQuery y hay una nueva herramienta open-source de visualización que vale la pena conocer.

01

OpenAI + Pandas: análisis de datos con lenguaje natural

La nueva API permite hacer preguntas en español sobre un DataFrame y obtener resultados automáticamente. Esto cambia bastante el flujo de trabajo para analistas junior que todavía están aprendiendo la sintaxis.

02

Google BigQuery reduce precios un 30% para queries bajo 1TB

Buena noticia para equipos chicos y freelancers. El cambio entra en vigor el 1 de mayo y aplica a todas las regiones. Una señal de que la guerra de precios en cloud data warehouses sigue.

03

Evidence.dev: reportes de datos con Markdown y SQL

Una herramienta open-source para crear dashboards estáticos directamente desde archivos .sql. Sin servidores, sin configuración. Ideal para analistas que quieren compartir insights rápido.

04

El rol de "Analytics Engineer" ya supera al "Data Analyst" en búsquedas en LinkedIn

Un cambio en la demanda del mercado que viene acelerando desde 2023. Si estás buscando trabajo, vale la pena entender qué implica este rol y si encaja con tu perfil.
